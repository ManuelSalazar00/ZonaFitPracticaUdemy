#  ZonaFit - Customer Management (DAO Practice Project)

## Description

This project is a foundational practice application focused on Object-Oriented Programming (OOP) using Java.  

It is designed to reinforce core software development concepts such as:

- Object-Oriented Programming principles
- DAO (Data Access Object) design pattern
- Clean Code practices
- SOLID principles
- Layered architecture
- GitHub Flow workflow

The application manages customer records using a console-based interface and a MySQL database.

---

## Technologies Used

- Java 17
- Maven
- MySQL
- JDBC
- Git & GitHub
- GitHub Flow
